    CMake备忘录

    Chap0 Intro

    简而言之，CMake是一个编译工具，在UNIX平台上可用于生成makefile。所以常见的工具链就是cmake+make

    Chap1 基础语法

    cmake本身有很多指令，用CMD() 表示指令 cmake中指令大小写无关，但参数是大小写相关的。为了统一，一般都用大小字母表示。 变量赋值采用SET命令，eg. SET(VAR [VALUE] [CACHE TYPE DOCSTRING [FORCE]]) 变量引用采用${}符号，但在if控制语句中直接使用变量名 cmake清理工程只需用make clean把make出的文件清理就行。 内部构建和外部构建：内部构建指在源代码所在目录去用cmake生成文件和make编译，外部构建是指在外部目录使用cmake及make编译。显然外部构建更有利于保持文件结构的整洁，故往往采用在build文件夹中使用cmake和make编译，把编译中间文件存于build中，方便管理

    Chap1 编译相关

    1. 一个工程就由一个PROJECT(name)来定义，工程会包含几个全局变量：
      • PROJECT_BINARY_DIR或<projectname>_BINARY_DIR用于表示编译发生的当前目录。
    2. 工程下每一个目录中都需要写CMakeLists.txt
    3. 如何包含子目录中的源文件？ ADD_SUBDIRECTORY命令： ADD_SUBDIRECTORY(source_dir [binary_dir] [EXCLUDE_FROM_ALL]) 用于向工程中添加存放源文件的子目录 。可选参数bianry_dir表示中间二进制文件和目标二进制文件存放的位置。标识参数EXCLUDE_FROM_ALL表示是否把这个目录从编译过程中排除。
    4. 输出结果位置 通过设置EXECUTABLE_OUTPUT_PATH来设置最终可执行文件所在位置。 通过设置LIBRARY_OUTPUT_PATH来指定最终共享库所在的位置。
    5. 安装问题
      1. 用make install中DESDIR makefile里面有变量DESDIR指定目标安装位置，默认装在$(DESDIR)/usr/bin目录下，所以使用命令make install DESDIR=/path/to/install可以指定安装位置
      2. cmake中的CMAKE_INSTALL_PREFIX变量 使用方法：cmake -DCMAKE_INSTALL_PREFIX=/path/to/insatll
      3. cmake中的INSTALL命令 INSTALL命令用于在cmake内部定义安装的一些配置 安装目标文件： cmake INSTALL(TARGETS targets... [[ARCHIVE|LIBRARY|RUNTIME] [DESTINATION] <dir> [PERMISSIONS] permissions... [CONFIGURATIONS [Degug|Release|...]] [COMPONENT <component>] [OPTIONAL] ][...] ) TARGETS指定ADD_EXECUTABLE和ADD_LIBRARY的目标文件，可以是可执行二进制、动态库、静态库，应该从文件名可以区分 后面的三大配置具体指明各种文件安装细节 分类：ARCHIVE静态库，LIBRARY动态库，RUNTIME可执行二进制 DESTINATION指定该文件安装位置，注意当路径以/开头，说明是一个绝对路径，这个路径会覆盖CMAKE_INSTALL_PREFIX。而是一个相对路径时，最终的安装路径是CMAKE_INSTALL_PREFIX/<DESTINATION> 安装一般文件： 类似操作，把TARGETS关键字改为FILES，默认权限为644 非目标文件安装（例如脚本）: 关键字为PROGRAMS，默认权限755 目录的安装： DIRECTORY参数指定需要安装的目录，注意dirs的结尾是abc和abc/有区别，abc表示把整个目录安装过去，也就是安装结果于/path/to/install/abc这个目录中。而abc/表示把这个目录中的内容安装到目标路径，而不包含目标本身，也就是说把abc目录中的内容全部安装到/path/to/install这个目录下。 而PATTERN怎是用正则表达式进行过滤，并把过滤后的文件执行EXCLUDE操作或者设置PERMISSIONS操作
      4. 安装时执行cmake脚本 INSTALL([SCRIPT <files>] [CODE <code>]) 其中script指定cmake脚本，code指定cmake指令
